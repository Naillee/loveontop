## 项目状态

`loveonTOP` 是基于 `daily_stock_analysis` fork 的演进版本，当前处于第一阶段品牌替换与产品方向重构中。

当前目标：

- 保留上游已有的多数据源、AI 分析、Web 工作台和消息推送能力
- 逐步重构为更贴近中文用户的 `涨停宝` 产品体验
- 后续补强股票短线场景、ETF 与基金能力、涨停复盘与机会追踪

## 当前基础能力

- 多市场行情与新闻聚合
- AI 决策报告与问股能力
- Web 工作台
- GitHub Actions / Docker / 本地运行
- 企业微信 / 飞书 / Telegram / Discord / Slack / 邮件推送

## 品牌说明

## 开发说明

当前仓库仍在吸收上游结构，文档、默认文案、界面展示名和配置项会逐步替换为 `涨停宝 / loveonTOP`。

在完成第一轮重构前，部分目录、脚本名、截图和文档链接仍可能保留上游痕迹，这是预期中的过渡状态。
